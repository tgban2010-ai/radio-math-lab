# radio-math-lab

Learning mathematics through real-world signals, FFT, SDR, and radio astronomy experiments.

## What I'm Exploring
- Fourier Transform (FFT)
- Signal processing
- SDR (Software Defined Radio)
- Hydrogen line detection
- Mathematical intuition through experiments

## Repository Structure
- `notebooks/` — Jupyter notebooks for experiments and learning
- `data/` — small sample datasets
- `images/` — plots and screenshots
- `notes/` — observations and ideas

## Current Focus
This project helps me build math understanding by connecting equations and code to actual signals and measurements.

## Tools
- Python
- Jupyter Notebook
- NumPy
- Matplotlib
- SDR hardware

## Goal
Build a personal math-and-signal lab that grows from algebra and FFT intuition toward deeper signal processing, physics, and radio astronomy.
